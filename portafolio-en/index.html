Technological skills

+

HTML 5

Its acronyms come from 'HyperText Markup Language'. This markup language allows us to create the basic structure (or body) of a website through tags. The number five (5) refers to the version in which is currently. All the titles, paragraphs, forms, videos, among many more things we have them thanks to this wonderful language.

+

CSS 3

CSS it is a style language. Its acronyms also have a meaning 'Cascading StyleSheet'. Surely you'll know what the number three means (3), true? Exact! Its current version. This language of styles allows us... well, to give styles to the body of our website. It's HTML makeup. All the design of a site is directly related to this language.

+

JavaScript

Blessed JavaScript! It's a programming language that allows us to add, remove, apply animations or effects to the components of our site. It is the perfect tool to make dynamic and very attractive websites linked to CSS and HTML. Over time it has evolved exponentially and right now it's absolutely necessary to know if you want to be a web developer.

+

Python

Python is a high-level programming language known for its simple and readable syntax. It is widely used in web development, scientific research, data analysis, artificial intelligence, and more. Its focus on code readability and productivity makes it a popular choice for beginners and experienced developers alike.

+

Java

Java is an object-oriented programming language designed to be portable, secure, and robust. It is used in a wide variety of applications, from enterprise application development to mobile application development (Android). Java runs on a virtual machine (JVM), making it platform-independent.

+

PHP

PHP is a general-purpose programming language specially suited for web development. It is widely used for creating dynamic websites and web applications. PHP integrates easily with HTML and runs on the server, generating web content in real-time. It is known for its ease of use and widespread availability on web servers.

+

WordPress

WordPress is a widely used content management system (CMS) for creating and managing websites. It is known for its ease of use and flexibility, making it a popular choice for both beginner users and advanced developers. WordPress offers a wide variety of themes and plugins that allow customization and expansion of website functionality.

+

SQL

SQL (Structured Query Language) is a query language used to manage and manipulate relational databases. It allows users to perform operations such as inserting, updating, deleting, and querying data in a database. SQL is fundamental for data management in a wide range of applications, from websites to enterprise systems.

+

Git

Git is a widely used distributed version control system for collaborative software development. It enables development teams to track changes in the source code, collaborate on projects, and manage versions efficiently. Git facilitates collaboration among developers and provides a comprehensive history of the changes made to a project.
