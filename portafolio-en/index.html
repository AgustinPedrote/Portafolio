HTML 5

HTML is a markup language used to create the structure of websites using tags. The current version is number five (5) and allows for the creation of various elements such as headings, paragraphs, forms, and videos.

+

CSS 3

CSS, which stands for "Cascading Style Sheets," is a styling language that provides visual design to websites, making it essential for their appearance. Its current version is number three (3).

+

JavaScript

JavaScript is essential for creating dynamic and attractive websites, allowing the addition of animations and effects. It is fundamental in modern web development and necessary for developers.

+

Git

Git is a version control system for collaborative software development. It allows tracking changes, collaborating efficiently, and managing versions, facilitating collaboration among developers and maintaining a complete history of changes.

+

Python

Python is a simple and readable programming language used in web development, data science, and more. It is popular for its ease of use and productivity.

+

Java

Java is an object-oriented programming language, portable, secure, and robust, used in various applications such as enterprise and mobile development (Android). It runs on a virtual machine (JVM), making it platform-independent.

+

PHP

PHP is a programming language for web development, integrates with HTML, and runs on the server to create dynamic websites. It is known for its ease of use and is widely available on web servers.

+

WordPress

WordPress is a popular CMS for creating and managing websites, known for its ease of use and flexibility. It offers a wide range of themes and plugins to customize sites according to needs.
